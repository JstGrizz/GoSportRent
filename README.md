## List Fitur

## 1. Admin

- **Login**
- **Manage Users**
  - View user list
  - Edit user (name, username, email, role)
  - Add, edit, delete users
- **Manage Units**
  - View unit list
  - Edit unit (name, category, stock, cost rent per day/month, image)
  - Add new units
- **Master Category**
  - View category list
  - Edit category (name)
- **Rental History**
  - View rental history
  - Edit history
- **Policy Management**

## 2. User

- **Login**
- **Update Profile**
- **My Rent**
  - Pay, return, pay fee (checks for over-limit policy)
- **Browse Units**
  - View units, rent (validate ongoing rentals, per day/month, unit count, rental duration)
